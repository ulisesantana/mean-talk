  - Todas las herramientas de desarrollo web moderno usan NodeJS y npm para construir las aplicaciones que acaban en producción. Aparte de las herramientas que existen para asistir el desarrollo.

</Notes>

# API REST

|  GET |  POST |     PUT    | DELETE |
|:----:|:-----:|:----------:|:------:|
| Leer | Crear | Actualizar | Eliminar |


<Notes>

  - MEAN no es sólo un stack tecnológico, sino también un conjunto de arquitecturas. Entre ellas RESt API y SPA.

  - Comentar que las decisiones de arquitectura deben ser enfocadas a solucionar un problema. No te pongas con microservicios a lo bestia si vas a hacer una app chorra que sólo vas a usar tú.
  
</Notes>

---
<List>

  - GET `/tasks/:userId`: Obtener todas las tareas de un usuario.
  - POST `/tasks`: Crear una nueva tarea.
  - PUT `/tasks/:userId/:taskId`: Actualizar la tarea de un usuario.

</List>
